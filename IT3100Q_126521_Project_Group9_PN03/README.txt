Để phần mềm được hoạt động, ta cần phải cài JDK và JRE.

Bước 1: Tải và cài đặt JRE
Linh tải JRE cho Windows: https://java.com/en/download/windows_offline.jsp
Nhấn vào Agree and Start Free Download để tải về và ghi nhớ thư mục đã lưu.
Sau khi tải xong nhấn đúp chuột vào jre-8u251-windows-i586.exe để cài đặt.
Bước 2: Tải và cài đặt Java JDK để lập trình Java
Tải file cài đặt Java từ trang chủ của Oracle.
Link tải Java JDK: https://www.oracle.com/java/technologies/javase-downloads.html#JDK14
Phiên bản được chọn là Java SE 14. Nhấn vào tải JDK Download.
Nếu muốn tải phiên bản Java JDK cụ thể nào thì cũng hoàn toàn ổn. Hoặc an toàn nhất là chọn phiên bản có ghi ký hiệu LTS (Long Term Support – Hỗ trợ dài hạn)
Để cài đặt được cho Windows 10 thì tải phiên bản Windows x64 installer. Nhấn vào biểu tượng tải xuống để tải về và cài đặt.
Tiếp tục nhấn tích vào “I reviewed and accept the Oracle Technology Network License Agreement for Oracle Java SE” .
Sau đó nhấn Download jdk-14.0.1_windows-x64_bin.exe
Lưu vào thư mục bất kỳ và nhớ tên thư mục để cài đặt.
Sau khi tải xong, nhấn đúp chuột vào file jdk-14.0.1_windows-x64_bin.exe để cài đặt.
Nhấn Next và tiếp tục.
Trình cài đặt sẽ hỏi nơi lưu phần mềm, có thể mặc định là C:\Program Files\Java\jdk-14.0.1\
Nhấn Close để hoàn tất việc cài đặt 

Kiểm tra thử xem Java đã được cài đặt thành công chưa bằng cách như sau:
Vào Search (hoặc nhấn Windows + R) tìm kiếm CMD và chạy nó. Sau đó nhấn câu lệnh sau: java -version

Các thư viện chúng em có sử dụng trong bài.
